1. Текущая конфигурация проекта
--------------------------------
Тема проекта:
«Сравнение нейронных сетей в задаче поиска сходных постпрандиальных CGM-окон».

Тип проекта:
ретроспективный исследовательский прототип для comparative neural retrieval.

Источник данных:
- используется соседний каталог `data`
- формат: XML-файлы `*-ws-training.xml` и `*-ws-testing.xml`
- число пациентов: 12

Режим по умолчанию:
- Память Хопфилда

Доступные top-level режимы:
- Память Хопфилда:
  ассоциативная память с итеративным восстановлением состояния и энергетической
  диагностикой.
- Сиамская temporal-модель:
  нейросетевая модель метрического пространства для поиска сходных окон.
- Карта Кохонена:
  карта самоорганизации для топологического neighborhood-based retrieval.

Общие данные по текущему срезу:
- Пациенты: 12
- Всего выделенных окон: 2168
- Окна в usable retrieval-срезе: 830
- Train-memory окон: 576

Распределение исследовательских меток:

- Контролируемый отклик: 193
- Неоднозначный отклик: 97
- Позднее снижение: 50
- Нестабильный отклик: 11

Причины исключения окон:
- Перекрытие со следующим приемом пищи: 510
- Перекрытие с предыдущим приемом пищи: 177
- Исходное состояние до начала окна: 378

Safety framing:
- система не является медицинским изделием;
- не формирует клинические рекомендации;
- не предназначена для выбора дозы инсулина;
- результаты нужно трактовать как сравнение retrieval-подходов, а не как
  доказательство клинической применимости.

5. Страница «Методология» (/about)
----------------------------------
Основной вопрос страницы:
«Как устроен пайплайн, какие модели сравниваются и какие ограничения у работы?»

На странице показываются:
- краткое введение о comparative neural retrieval;
- разделы:
  1. Задача
  2. Данные OhioT1DM
  3. Выделение постпрандиальных окон
  4. Признаки
  5. Модели
  6. Метрики
  7. Ограничения
- базовые формулы для Hopfield, Siamese retrieval и SOM BMU.

Содержательная интерпретация страницы:
- Память Хопфилда:
  associative recall над train-векторами признаков.
- Сиамская temporal-модель:
  learned metric space с nearest-neighbor retrieval.
- Карта Кохонена:
  topology-preserving self-organization для локальных neighbourhoods.

Ограничения, которые нужно держать в тексте summary:
- исследование всё ещё остаётся малой ретроспективной выборкой;
- метки имеют исследовательский и детерминированный характер;
- similarity не равна клинической рекомендации;
- умеренные retrieval-результаты не следует трактовать как клиническую валидацию.
